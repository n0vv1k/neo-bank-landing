**NEOBANK** — это концептуальный лендинг для финансовой экосистемы будущего. Дизайн выполнен в премиальной стилистике "Silver & Dark", с акцентом на иммерсивный пользовательский опыт, плавные анимации и высокотехнологичный визуал.

## ✨ Ключевые особенности (WOW-эффекты)

Проект демонстрирует современные подходы к фронтенд-разработке:

- **3D Hero Section:** Интерактивная банковская карта с использованием CSS 3D-трансформаций и левитацией на GSAP.
- **Bento Grid 2.0:** Адаптивная сетка функционала с эффектом "Spotlight" (свечение, следующее за курсором).
- **Immersive Parallax:** Кинематографичная секция с глубоким эффектом параллакса при скролле.
- **Dynamic Counters:** Анимированные счетчики статистики, срабатывающие при достижении секции.
- **Fluid Typography:** Использование CSS-функции `clamp()` для идеально плавной адаптивности без лишних медиа-запросов.

---

## 🛠 Технологический стек

- **Frontend:** HTML5, CSS3 (Custom Properties, Grid, Flexbox).
- **Animations:** [GSAP (GreenSock)](https://greensock.com/gsap/) + ScrollTrigger.
- **Typography:** Inter / System UI.
- **Design Pattern:** Glassmorphism, BEM Methodology.
